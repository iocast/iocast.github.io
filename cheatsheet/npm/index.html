    MacOS sheat sheet

    prepublishOnly ensures that the library is always built before we publish to npm.

    more flags here https://docs.npmjs.com/misc/scripts

    needed dev libraries

    babel-cli provides the command babel. babel-preset-env is the Babel preset we use for transpilation.

    for tests use babel-register lets AVA execute the tests via Babel.
