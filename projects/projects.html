Open Source Projects

featureserver

is an implementation of a RESTful Geographic Feature Service using standard HTTP methods.

vectorformats

is a library for easy serialization and deserialization of geographic features into various formats.

i18n plugin for bottle

integrates the multilingual internationalization services gettext from Python with your Bottle application.

hook server

is a Python server which updates a git repositories triggered by a post hook comming from GitHub or Bitbucket.

server management

holds a collection of scripts to automatically backup databases, such as PostgreSQL or MariaDB, to migrate a Windows or OS X file server and backup files to a SMB share.
